# VoyageFlow .gitignore

This file excludes common development and build artifacts from version control.

## Exclusions:
- **Node.js dependencies**: /node_modules, .pnp files
- **Testing**: Coverage reports
- **Next.js**: Build outputs and production files
- **System files**: macOS .DS_Store, SSL certificates
- **Debug logs**: npm and yarn debug files
- **Environment**: Local environment variables
- **Deployment**: Vercel-specific files
- **TypeScript**: Build artifacts and type definitions
