# **App Name**: VoyageFlow

## Core Features:

- Secure User Authentication: Enables user login and registration with email/password and social options (Google, Facebook), ensuring secure access to the platform.
- Travel Service Search: Provides an interface for users to search and filter various travel services like flights, hotels, and activities.
- Booking Basket Management: Allows users to add, review, and manage selected travel options in a temporary basket before checkout.
- Real-time Availability Verification: Integrates with external APIs to check and confirm the real-time availability of selected travel services prior to booking.
- Streamlined Checkout & Payment: Facilitates a secure checkout process, integrated with Paystack for seamless and reliable payment processing.
- Booking Confirmation & Reference: Generates a unique booking reference and displays confirmation details immediately after successful payment.

## Style Guidelines:

- Primary brand color: A deep, inviting blue (#2A6CF5) to evoke trust and the spirit of adventure, contrasting effectively with the light background.
- Background color: A very light, subtle bluish-grey (#F5F7F8), providing a clean and open canvas, aligning with a light theme for clarity and modern aesthetics.
- Accent color: A vibrant sky blue (#33B2E6), chosen as an analogous hue to the primary, offering a dynamic highlight for calls to action and important interactive elements.
- Body and headline font: 'Inter' (sans-serif) for its modern, highly legible, and versatile qualities, suitable for both display and extended text. Note: currently only Google Fonts are supported.
- Utilize modern, clean line icons related to travel, exploration, and user actions, maintaining consistency across the application for an intuitive experience.
- A responsive, modern layout with a clear visual hierarchy, utilizing the 'glass-effect' for card-like elements and ample spacing to create an elegant and spacious user interface, as demonstrated in the provided HTML skeleton.
- Subtle and smooth transition effects for interactive elements like buttons and tabs, enhancing user feedback and contributing to a fluid navigation experience, as hinted by CSS 'transition-colors'.
